# Configuring Vertex AI Service Account

## Step A. Configuring Your Vertex AI Service Account

1. Open the Google Cloud Console via the **Cloud Resources Tab** above.
2. Go to **IAM & Admin > Service Accounts**.
3. Click **Create Service Account**. Name it `vertex-agent-user`.
4. Important: Grant it the role **Vertex AI User**.
5. Click **Done**.
6. Click on your new service account in the list, go to the **Keys** tab, and click **Add Key > Create new key**.
7. Select **JSON** and the file will download to your _physical computer_.

## Step B: Upload to Workspace (If you are Using Udacity Workspace)

1. Go back to your VS Code Workspace.
2. Drag and drop that JSON file from your computer into the file explorer pane (inside the `project-starter` folder).
3. Rename it to something simple, like `service-account-key.json`.

## Step C. Configure the Environment Variables

Now we tell the code where to find that key.

1. In the file explorer, find `.env.example`.
2. Right-click and rename it to `.env`.
3. Update the path to match the file you just uploaded:

```txt
GOOGLE_APPLICATION_CREDENTIALS=service-account-key.json
PROJECT_ID=your-actual-gcp-project-id
```

> Note: You can find your `PROJECT_ID` on the dashboard of the Google Cloud Console.

## Step D. Verification

To make sure Vertex AI is connected, run the main script:

```sh
python main.py
```

If you see the console print `✓ Client initialized with Vertex AI`, you are ready to build!
